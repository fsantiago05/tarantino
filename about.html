Sobre o Projeto

Bem-vindo ao Universo Tarantino! Este projeto é uma homenagem à filmografia de Quentin Tarantino, apresentando uma lista interativa de seus filmes icônicos.

Funcionalidades

  • Listagem de Filmes: Exibe cards com detalhes dos filmes (título, ano, sinopse e pôster).
  • Busca Interativa: Permite filtrar os filmes pelo nome na página principal.
  • Links para Trailers: Ao clicar em um card, o usuário é redirecionado para o trailer do filme no YouTube.
  • Design Responsivo: Layout adaptável para diferentes tamanhos de tela.

Tecnologias Utilizadas

  • HTML5: Estrutura semântica da página.
  • CSS3: Estilização e layout (Flexbox/Grid).
  • JavaScript: Lógica para carregar dados do JSON e funcionalidade de busca.
  • JSON: Armazenamento dos dados dos filmes.
